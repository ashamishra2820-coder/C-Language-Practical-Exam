Overview

This repository contains solutions to the C Language Practical Examination. The programs demonstrate the use of functions, arrays, pointers, conditional statements, and nested loops in C programming.

Program 1: Sum of First and Last Digit
Description

This program accepts a 3-digit positive integer from the user and calculates the sum of its first and last digits.

Output
Enter a 3-digit positive integer: 282

First Digit = 2
Last Digit = 2
Sum = 4
Program 2: Count Occurrences of Consonants
Description

This program reads a sentence from the user and counts the frequency of each consonant present in the sentence.

Output
Enter a sentence: hello world

d = 1
h = 1
l = 3
r = 1
w = 1
Program 3: Check Divisibility by 3 and 5
Description

This program checks whether a given number is divisible by both 3 and 5 using a function.

Output 1
Enter a number: 4

4 is not divisible by both 3 and 5.
Output 2
Enter a number: 15

15 is divisible by both 3 and 5.
